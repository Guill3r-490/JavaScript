Operadores

Aritiméticos

São usados para fazer calculo:

+ > usado para soma

- >usado para subtração

* > usado para mutiplicação

/ > usado para divisão

** > usado para potenciação

% >usado para divisão tirando a vírgula, sendo o resto da divisão

Atribuição

= ->usado para atribuir valores (recebe)

auto-atribuições: var n=3 -> n+=4 ->n=7. Outra forma de fazer n+=1 é ++n ou n++

Relacionais

> maior que

< menor que

>= maior ou igual a

<= ou igual a

== sinal de igual

!= sinal de diferente

=== pergunta se são indênticos

Lógicos

! negação, não

&& conjução, e

|| disjunção, ou

Ordem de execução: ! > && > ||

Ternário

?

:

Exemplo: teste ? oque vai acontecer se for true ; oque vai acontecer se for false

Aplicação: média>= ?'Aprovado' ;'Reprovado'
